# Contributing

Could You Make is a personal project that I run for my own apps and share publicly so others can see how it's built.

**Issues are welcome.** If you spot a bug, a security problem, or something in the README that's wrong or unclear, please open one. Reproduction steps are appreciated for bugs; a short rationale is appreciated for feature suggestions.

**Pull requests are not actively solicited.** I'm happy to look at small, focused PRs (typo fixes, obvious bug fixes, documentation improvements), but I may decline larger changes that don't fit how I use the project. If you have something substantial in mind, open an issue first so we can talk through whether it's a good fit before you spend time on it.

**Forking is encouraged.** If you want to take this in a different direction — multi-tenant, attachments, a different stack — fork it. The MIT license is permissive and you don't need my permission.

**Security issues** should be reported privately via GitHub's "Report a vulnerability" link on the Security tab rather than as a public issue.
